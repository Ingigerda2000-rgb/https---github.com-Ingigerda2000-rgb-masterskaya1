# Исправление названий шагов в конструкторе индивидуальных заказов

## Задача
В конструкторе индивидуальных заказов при переходе в настройки название каждого этапа отображается на английском языке.

## Выполненные действия

### ✅ Завершено
- [x] Анализ проблемы: Названия шагов в прогрессе показывались на английском (например, "Configuration" вместо "Настройка")
- [x] Обновление views.py: Добавлен словарь с русскими названиями шагов и переменная step_display_name в контекст
- [x] Обновление всех шаблонов конструктора: Заменено {{ step_name|capfirst }} на {{ step_display_name }} в 6 файлах:
  - constructor_configuration.html
  - constructor_category.html
  - constructor_product.html
  - constructor_description.html
  - constructor_options.html
  - constructor_summary.html

### ✅ Результат
Названия шагов в конструкторе теперь отображаются на русском языке:
- Категория (вместо Category)
- Товар (вместо Product)
- Настройка (вместо Configuration)
- Описание (вместо Description)
- Дополнительно (вместо Options)
- Итог (вместо Summary)
